# 麻将连连看

基于 HTML/CSS/JS 的麻将连连看浏览器游戏，无需安装，打开即玩。

## 玩法

点击两张相同且路径畅通（0~2 折）的麻将牌即可消除。消除全部牌获胜。

## 关卡

| 关卡 | 矩阵 | 牌种 | 涵盖类别 |
|------|------|------|----------|
| 入门 | 6×8 | 12 种 | 筒子 |
| 简单 | 8×10 | 20 种 | 筒子、索子 |
| 进阶 | 8×12 | 24 种 | 筒子、索子、万子 |
| 挑战 | 10×12 | 30 种 | 筒子、索子、万子、番子 |
| 困难 | 10×14 | 35 种 | 筒子、索子、万子、番子、花牌 |
| 大师 | 10×16 | 40 种 | 全类别 |

随机配方保证每局体验略有不同。

## 功能

- 提示、洗牌、主题切换（经典/动物/节庆）
- 连击系统：3x 以上闪电连线 + Combo 特效
- 死局自动洗牌
- 窗口自适应缩放
- 深色星空背景

## 运行

用浏览器打开 `index.html` 即可。
